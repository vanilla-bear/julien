BUEditor
http://drupal.org/project/bueditor
====================================


DESCRIPTION
-----------
BUEditor is a text editor that can be used for various markup languages like HTML, Textile, Markdown, etc.


INSTALLATION
-----------
1- Copy bueditor directory to your modules(sites/all/modules) directory.
2- Enable the module at /admin/modules.
3- Manage editors at /admin/config/content/bueditor.
4- Associate editors with text formats at /admin/config/content/formats.
